# Chat App

## Descripción
Aplicación de chat construida con Vue.js y Node.js.

## Requisitos
- Node.js
- MongoDB

## Instalación

1. Clona el repositorio:
   ```bash
   git clone <tu-repositorio-url>

2. Navega a la carpeta del servidor y ejecuta:
    ```npm install
    nodemon app.js

3. Navega a la carpeta del cliente y ejecuta:
    ```npm install
    npm run serve
